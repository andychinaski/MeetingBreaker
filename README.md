# Meeting Breaker

Meeting Breaker — юмористическая браузерная игра в жанре Arkanoid о борьбе за свободное время в календаре IT-компании. Игрок управляет платформой **Focus Time**, отбивает шар-задачу **Task** и разрушает блоки встреч, стилизованные под рабочую неделю.

> MVP реализован: доступен один законченный уровень, главное меню, настройки, бонусы, локальные рекорды, победа и поражение.

## Идея

Календарь используется только как визуальная метафора — интеграций с Outlook, Google Calendar и другими сервисами не предусмотрено. Игра работает полностью в браузере, без backend, авторизации и передачи персональных данных. Настройки и рекорды хранятся локально.

Основной игровой цикл:

1. Игрок запускает шар с платформы.
2. Шар отражается от стен, платформы и блоков встреч.
3. Встречи теряют прочность и разрушаются, принося очки и «освобождённое время».
4. Потеря шара расходует одну чашку кофе.
5. Уровень заканчивается победой после уничтожения обязательных встреч или поражением, когда кофе закончился.

## Возможности MVP

- календарная сетка рабочей недели и 21 встреча восьми типов;
- разная прочность встреч, очки, свободное время и комбо-множитель;
- три чашки кофе вместо жизней;
- пауза, перезапуск, победа и поражение;
- светлая и тёмная темы, громкость и отключение звука;
- локальное сохранение настроек и рекордов;
- процедурные звуки без внешних аудиофайлов.

Целевая платформа — современные desktop-браузеры. Минимальное поддерживаемое разрешение: `1366×768`.

## Управление

| Действие | Клавиатура | Мышь |
| --- | --- | --- |
| Движение платформы | `A` / `D` или `←` / `→` | движение курсора |
| Запуск шара | `Space` | левый клик |
| Пауза | `Esc` | — |
| Перезапуск уровня | `R` | — |

## Бонусы

- **Async Mode** — добавляет второй шар; кофе расходуется только после потери всех шаров.
- **Decline** — следующее попадание мгновенно уничтожает любую встречу.
- **Espresso Shot** — на 10 секунд увеличивает платформу и снижает скорость шаров; повторный подбор обновляет таймер.

## Стек

- TypeScript и React — интерфейс, меню, HUD и настройки;
- Vite — сборка и локальная разработка;
- Phaser 3 — игровая сцена, физика и эффекты;
- Vitest — модульные и интеграционные тесты;
- Playwright — end-to-end тесты;
- `localStorage` — настройки и рекорды.

## Ограничения MVP

- нет backend, авторизации и сетевой игры;
- нет интеграций с Outlook, Google Calendar и реальными календарями;
- поддерживаются современные desktop-браузеры от `1366×768`;
- доступен один уровень, мобильная версия и редактор уровней не входят в MVP;
- игровой прогресс активной сессии не сохраняется.

## Запуск

```bash
npm install
npm run dev
```

Vite выведет адрес локального сервера в терминал. Для остальных проверок доступны команды:

```bash
npm run test
npm run lint
npm run build
npm run test:e2e
```

Для первого запуска end-to-end тестов может потребоваться установка Chromium командой `npx playwright install chromium`.

## Структура

```text
src/
├── app/          # React-приложение

tests/e2e/        # Playwright-тесты
```
