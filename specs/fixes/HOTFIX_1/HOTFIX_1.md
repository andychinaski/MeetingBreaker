# HOTFIX 1

## Задачи

1. Заменить:

```text
Ваш календарь сопротивляется
```

на:

```text
Дай календарю пизды
```

2. Увеличить вертикальный отступ между заголовком `Meeting Breaker` и подзаголовком:

```text
Освобождайте фокус-время — по одной встрече за раз.
```

3. Во всех пользовательских текстах заменить формулировки про рабочий день на формулировки про рабочую неделю.

Минимум:

```text
Рабочий день завершён
```

заменить на:

```text
Рабочая неделя завершена
```

4. Проверить главный экран, экраны победы/поражения, модальные окна, конфиги и текстовые константы.

5. Выполнить:

```bash
npm run lint
npm run build
```

Не менять игровую логику и дизайн за пределами указанных правок.
